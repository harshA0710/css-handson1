What is CSS?

  1. CSS stands for Cascading Style Sheet
  2. CSS describe how the HTML element are to be displayed on screen,paper or in other media
  3. CSS is not neccessaryCSS saves a lot of workIt can control the layout of different web pages all at ones.
  4. External stylesheet are stored in CSS files

CSS can be added to HTML documents in three ways:

  • Inline - by using style attribute inside HTML element1

  • Internal - by using a style element in the head section

  • External - by using a link element to link to an external file

  • The most common way to add css is to keep the styles in external CSS file. However in this tutorial we use inline and internal styles because it is easier the demonistrate and easier gor you to try to yourself

When comparing CSS classes and ID, the differences is that css class can apply to multiple elements. ID on the other hand, applies css to a unique elements. ID is also speial in that you can use a special url to link directly to an elements and it is used by JavaScript


In CSS, selector are used target a specificelements or range of element on a web page. Once a elements is being targeted, a style or set of styles can be applied to an elements.


There are wide range of selector available. Two of the most commomly used are class and ID. Both are used to target elements to which a style should applied
